flask_routes.flask_helper_functions.format_for_endpoint(db_dictionary, success_scenarios_array)

Takes a db dict and returns a formatted dict for delivery to endpoint.

Returns Dictionary with keys:

<status> API_SUCCESS_STRING, API_FAILURE_STRING or API_ERROR_STRING.

Error: When ‘result’ contains a value in flask_helper_functions.error_scenarios. Success: When ‘result’ contains a value in success_scenarios_array. Failure: When ‘result’ does not contain a value in success_scenarios_array.

<message> ‘result’’s associated value.

<reason> (if <status> is API_FAILURE_STRING) [Array] Reason for failure.

<data> (if <status> is API_SUCCESS_STRING) [Array] ‘content’ OR None.
